    Android Development
    GDG Bambili supports developers with resources for learning and building apps on the android platform. These resources show how to accomplish a specific task with code samples you can re-use in your app.
    Google Cloud Platform
    Modernize your workloads on world-class infrastructure, by migrating quickly with pre-packaged cloud infrastructure solutions. Protect your data with multilayered security, Secure-by-design infrastructure protects your data, applications, and users, with advanced anti-malware and threat detection.
    Machine Learning with Tensorflow
    TensorFlow offers multiple levels of abstraction so you can choose the right one for your needs. Build and train models by using the high-level Keras API, which makes getting started with TensorFlow and machine learning easy. If you need more flexibility, eager execution allows for immediate iteration and intuitive debugging. For large ML training tasks, use the Distribution Strategy API for distributed training on different hardware configurations without changing the model definition.
